# Dark Dracula

A dark VS Code color theme that combines the iconic IntelliJ-ish Darcula syntax highlighting with the sleek, modern editor and UI colors from VS Code's built-in 2026 Dark theme.

## What's Inside

- **Syntax highlighting** from [IntelliJ-ish Darcula](https://marketplace.visualstudio.com/items?itemName=csantiago132.intellij-ish-darcula-theme) — the warm golds, greens, and purples that make code easy to read
- **Editor & UI colors** from VS Code's 2026 Dark — deep `#121314` editor background, subtle borders, and a cohesive dark chrome throughout

## Installation

1. Copy the extension folder into:
   ```
   C:\Users\<username>\.vscode\extensions\dark-dracula\
   ```
2. Reload VS Code (`Ctrl+Shift+P` → `Developer: Reload Window`)
3. Open `Ctrl+Shift+P` → `Preferences: Color Theme` → select **Dark Dracula**
